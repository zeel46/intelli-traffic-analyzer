Static Site Tracking Demo

Tracking Code Embedded!

If the backend server is running on port 5000, and you replaced YOUR_API_KEY_HERE in the source code with a real key from the database, a page view just got recorded!

How to test:

  1. Start your Node.js backend (npm run dev in the server folder).
  2. Open this file in your browser (just double-click test-site.html to open locally).
  3. Check the Network tab in Developer Tools (F12) to see the POST request to /api/track.
  4. You can also click around and refresh the page to generate more traffic data points!
